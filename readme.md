## Setup

1. Ensure you have Python 3.10 installed.
2. Create a virtual environment: `py -3.10 -m venv venv` (Windows) or `python3.10 -m venv venv` (Linux/Mac).
3. Activate the virtual environment: `venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Linux/Mac).
4. Install dependencies: `pip install -r requirements.txt`.
5. Open and run the notebooks in VS Code or Jupyter.
